# E-commerce System Skeleton

This is a simple skeleton for an e-commerce system implemented using Spring Boot, Spring Cloud, MySQL, MyBatis Plus, ShardingJDBC, and gRPC.

## Modules
- **Spring Boot** is used as the base framework.
- **Spring Cloud** can be used for service discovery or other cloud capabilities.
- **MySQL** is the database.
- **MyBatis Plus** is the ORM framework.
- **ShardingJDBC** provides database sharding capabilities.
- **gRPC** is used for communication between the frontend and the backend.

This project only contains minimal configuration and example code to show how the components can be integrated. It is not a complete production-ready e-commerce system.
